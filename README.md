# Propulsion Feed & Test Operations

Once evolved... make it a Propulsion Test Stand Instrumentation Mini-Rig

| Digital Twin Layer             | What it does                                                   | Common Tools                                                                     |
| ------------------------------ | -------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| **CAD / Geometry**             | Defines the physical asset geometry                            | SolidWorks, Siemens NX, CATIA, PTC Creo, Fusion 360                              |
| **PLM / Digital Thread**       | Tracks design versions, requirements, BOMs, manufacturing data | Siemens Teamcenter, 3DEXPERIENCE/ENOVIA, PTC Windchill                           |
| **MBSE / Requirements**        | Defines system architecture and interfaces                     | Cameo Systems Modeler, IBM Rhapsody, Capella, Ansys ModelCenter                  |
| **Physics Simulation**         | Predicts behavior before/while operating                       | Ansys Fluent/Mechanical/Twin Builder, STAR-CCM+, Abaqus, COMSOL, SimScale        |
| **System/Controls Modeling**   | Models actuators, sensors, controllers, dynamics               | MATLAB/Simulink, Simscape, Modelica, Dymola, Amesim                              |
| **Data Acquisition**           | Collects real sensor data                                      | NI LabVIEW, NI DAQ, Arduino/Raspberry Pi for student projects, PLC/SCADA systems |
| **IoT / Cloud Twin Platform**  | Connects live assets to cloud models                           | Azure Digital Twins, AWS IoT TwinMaker, PTC ThingWorx, Siemens Insights Hub      |
| **Data + ML**                  | Cleans data, detects faults, predicts future states            | Python, pandas, NumPy, SciPy, scikit-learn, PyTorch, MATLAB                      |
| **Visualization / Dashboards** | Shows health, performance, alerts, trends                      | Grafana, Power BI, Tableau, Plotly Dash, Streamlit                               |
| **3D/Immersive Twin**          | Visual interactive twin / factory / robot simulation           | NVIDIA Omniverse, Unity, Unreal Engine                                           |
